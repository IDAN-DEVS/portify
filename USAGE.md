## Installation

1. **System Requirements**

   - Node.js 16.8 or later
   - npm or yarn
   - Git

2. **Setup**

   ```bash
   # Clone the repository
   git clone https://github.com/JC-Coder/portify.git

   # Navigate to project directory
   cd portify

   # Install dependencies
   npm install

   # Start development server
   npm run dev
   ```
